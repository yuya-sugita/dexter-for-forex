# 日次トレードルーティン

## API制限への対応方針

Twelve Data無料プランは**1分間に8リクエスト**が上限。
各ステップ間で自動的にレート制限が適用されるが、以下のルールで効率的に進める：

- 1ステップにつき**1〜2ツール呼び出し**に抑える
- 各ステップの結果を**完了してから次へ進む**（並列呼び出しを避ける）
- ユーザーが指定した銘柄がなければ、**1〜2銘柄に絞る**
- マクロ分析は**API負荷が高い**ため、明示的に求められた時のみ実行

## ワークフロー

```
日次ルーティン（所要時間: 3〜5分）:
- [ ] ステップ1: 経済カレンダー確認（1 APIコール）
- [ ] ステップ2: 対象銘柄の統計レジーム判定（1 APIコール）
- [ ] ステップ3: ボラティリティレジーム確認（1 APIコール）
- [ ] ステップ4: z-score確認（1 APIコール）
- [ ] ステップ5: 金利差確認（APIコールなし — ローカルデータ）
- [ ] ステップ6: 総合判断とトレードプラン提示
```

## ステップ1: 経済カレンダー確認

**ツール:** `get_economic_calendar`（今日〜明日、importance: "high"）

**目的:** 今日のハイインパクトイベントを把握し、トレード可否を判断。

**判断:**
- HIGH影響イベントが4時間以内 → そのペアはエントリー不可と明記
- HIGH影響イベントが今日中にある → 注意を促す

**結果を待ってから次へ進む。**

## ステップ2: 統計レジーム判定

**ツール:** `get_return_distribution`（対象銘柄、interval: "1day", lookback: 100）

ユーザーが銘柄を指定していなければ、EUR/USDをデフォルトとする。
複数銘柄の場合は1つずつ順番に実行する。

**抽出:**
- Hurst指数 → トレンド/平均回帰/ランダムウォーク
- 歪度・尖度 → テールリスク
- 自己相関 → モメンタム/平均回帰の強さ

**結果を待ってから次へ進む。**

## ステップ3: ボラティリティレジーム

**ツール:** `get_volatility_regime`（対象銘柄、interval: "1day"）

**抽出:**
- LOW/NORMAL/HIGH/CRISIS判定
- ポジションサイジング調整値
- vol期間構造

**結果を待ってから次へ進む。**

## ステップ4: z-score確認

**ツール:** `get_zscore`（対象銘柄、interval: "1day", lookback: 100）

**抽出:**
- 現在のz-score → 平均からの乖離度
- パーセンタイルランク
- 平均回帰確率

**結果を待ってから次へ進む。**

## ステップ5: 金利差（ローカル）

**ツール:** `get_rate_differential`（ベース/クォート通貨）

これはAPIコールを使わないローカルデータなので即座に実行可能。

**抽出:**
- 金利差と方向
- キャリートレードバイアス

## ステップ6: 総合判断

すべてのデータを統合し、以下のフォーマットで提示：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
本日のトレードプラン
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📅 経済イベント
  [HIGH影響イベントのリスト / なければ「重要イベントなし」]

📊 対象銘柄: [銘柄名]
  統計レジーム: [TRENDING / MEAN_REVERTING / RANDOM_WALK]（H=X.XX）
  ボラティリティ: [LOW/NORMAL/HIGH/CRISIS]（Xパーセンタイル）
  z-score: [X.XXσ]（パーセンタイル X%）
  金利差: [+/-X.XX%]（[方向]）

🎯 本日の方針
  戦略タイプ: [モメンタム / 平均回帰 / 見送り]
  リスク/トレード: [X.X%]
  ストップ幅: [X.X × ATR]
  注意事項: [イベントリスク等]

  [統計的エッジが見つからない場合: 「本日は見送り推奨」と明記]
```

## 追加分析が必要な場合

ユーザーが以下を求めた場合は個別に対応（レート制限を考慮し1つずつ）：
- 「マクロレジームも見て」→ `get_macro_regime`（5 APIコール使用、約1分待機あり）
- 「クロスアセットは？」→ `get_cross_asset_regime`（4 APIコール使用）
- 「相関も確認して」→ `get_correlation_matrix`（銘柄数分のAPIコール）
- 「バックテストして」→ `backtest_strategy`（1 APIコール）
- 「モンテカルロを回して」→ `monte_carlo_simulation`（APIコールなし）
